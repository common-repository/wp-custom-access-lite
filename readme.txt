== Description ==

WP - Custom Access Lite is a wordpress plugin that enables you to manage access to your pages without the wordpress login system.
Custom Access Lite allows you to hide your content behind a login system, and manage who can access that content.
With Custom Access Lite you can manage users which functions like: adding users with a desired access level, edit users and deleting users.
We also provide you with a log system which logs activities on your website and warns you about malicious events.

== Installation == 

How to install this plugin:

1. Login on your Wordpress website.
2. On the left side menu, go to "Plugins".
3. Click on "Add new".
4. Type in the searchbar, in the top right corner, "Custom Access Lite".
5. Click on "Install Now".
6. In the "Plugins" tab, at the left side menu, go back to "Installed Plugins".
7. Find Custom Access Lite and click on "Activate".

== Screenshots ==

1. On the homescreen of the plugin, you'll find information about the plugin. Click on the "Installation information" button to find out how to use the plugin.
2. In the "Add users" screen, you can register users, edit user information and delete users.
3. In the "options" screen, you can fill in your google captcha keys, and set log options.
